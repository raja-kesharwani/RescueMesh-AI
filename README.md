# RescueMesh AI

**From chaos to coordinated rescue.**

RescueMesh AI is an AI-powered disaster response platform that helps emergency teams collect SOS reports, prioritize critical cases, assign responders, visualize affected zones, and coordinate resources during disasters.

## Problem

During floods, earthquakes, wildfires, hurricanes, and other emergencies, response teams receive scattered reports from many people at once. Important cases can be delayed because reports are hard to organize, prioritize, and route quickly.

## Solution

RescueMesh AI converts emergency reports into a live command system. Users submit SOS reports, and the platform estimates urgency, suggests required resources, detects possible duplicate reports, maps incidents, and generates a rescue action plan for response teams.

## Key Features

- SOS emergency report form
- AI-style triage: Critical, High, Medium, Low
- Resource matching for rescue teams, ambulances, shelters, supplies, and fire units
- Offline SOS mode using local storage
- Duplicate report detection
- Live emergency dashboard
- Responder assignment and ETA tracking
- Crisis map with priority-colored markers
- AI command center action plan
- Disaster simulator for demo scenarios
- Pitch page for hackathon presentation
